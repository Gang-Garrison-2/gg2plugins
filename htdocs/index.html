ganggarrison.com/plugins

This directory provides the source of "server-sent plugins" used in GG2. Despite the name, they are actually downloaded from this "trusted" source.

This directory's actual URL is http://gg2plugins.ajf.me/, however it is reverse-proxied by ganggarrison.com. So please, use http://ganggarrison.com/plugins/ instead.

Plugins can be proposed by pull requests to the repo on GitHub: https://github.com/TazeTSchnitzel/gg2plugins

Also see the forum thread: http://www.ganggarrison.com/forums/index.php?topic=33377
